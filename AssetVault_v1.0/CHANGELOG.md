# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-03-06

### Added
- **Core WPF Application:** Initial release of the AssetVault GUI using `PresentationFramework`.
- **WMI Diagnostics Engine:** Deep integration for CPU, Storage, RAM, and Battery data capture.
- **PDF Exporter:** Microsoft Edge Headless engine integration for converting HTML arrays to printed PDFs.
- **Reporting System:** Automated Time-to-Replacement calculations based on inputted purchase dates.
- **UAC Ready:** Shipped with a local Authenticode signature identifying Arbor Solutions as the Publisher.
