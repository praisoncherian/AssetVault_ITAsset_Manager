# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| v1.0.0  | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

AssetVault handles sensitive system and telemetry data via WMI paths. If you discover a vulnerability or a flaw that could expose system configuration data or permit Privilege Escalation due to the `RequireAdmin` PowerShell hooks, please report it immediately.

Do not open a public issue. Email the vulnerability details directly to the Arbor Solutions team. We will acknowledge receipt within 48 hours and provide a timeline for a patch.
