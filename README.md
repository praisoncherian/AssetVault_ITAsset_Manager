# AssetVault IT Asset Manager

![AssetVault Logo](AssetVault_app_icon.png)

AssetVault is a comprehensive IT Asset Management (ITAM) platform designed to capture, report, and manage system diagnostics and hardware inventory using a powerful PowerShell-driven WPF interface.

## 🚀 Features
- **Deep System Diagnosis:** Extract precise WMI data (CPU, RAM, Storage Health, BitLocker status).
- **Automated Lifecycle Math:** Calculate warranty, purchase age, and "Time to Replacement" triggers.
- **Dynamic PDF Reporting:** Generate fully styled off-board hardware terms & conditions into secure PDFs.
- **Self-Contained Executable:** Compiled natively using `ps2exe` into a portable, signed Windows binary.

## 📦 Download & Run
AssetVault v1.0.0 is available as a portable Windows Executable.
1. Download the latest `AssetVault_v0.1.zip` release.
2. Extract the contents.
3. Run `AssetVault.exe` (requires Administrator privileges for WMI queries).
